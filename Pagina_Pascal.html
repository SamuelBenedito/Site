O que é a linguagem Pascal?


É uma linguagem de programação idealizada em 1971 por Niklaus Wirth, professor da faculdade Politécnica de Zurique, na Suíça, que assim foi batizada em homenagem ao grande matemático Blaise Pascal. O objetivo era desenvolver uma linguagem de programação disciplinada para ensinar programação estruturada. No fundo, trata-se de uma versão simplificada para propósitos educacionais da linguagem Algol, que data de 1960.
