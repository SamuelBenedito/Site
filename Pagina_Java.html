O que é o Java?

Java é uma linguagem de programação orientada a objetos, e é uma das linguagens de programação mais utilizada pelas empresas na atualidade em desenvolvimento de aplicações WEB e mobile. O Java foi criado na empresa Sun Microsystem em 1995 por uma equipe chefiada por James Gosling, conhecido como pai do Java. Em 2008, o Oracle adquiriu o java e vem crescendo desde então no mercado de tecnologia.



Principais características do Java:

Algumas características que podemos citar sobre a Linguagem Java são, por exemplo:


  • Orientada a Objetos;
  • Independente de plataforma;
  • Simples;
  • Tipada;
  • Compilada;
  • Interpretada;
  • JDK (Java Development Kit);
  • JRE (Java Runtime Environment).

Mas, por que Java?

Como já foi dito, Java é uma das linguagens mais utilizadas no mundo, várias empresas continuam adotando a linguagem. Além de ter vários frameworks para que possa utilizar nas suas aplicações, como por exemplo o Spring, Quarkus, Micronauts, JSF, dentre outros.

Uma das grandes vantagens do Java é poder criar aplicações para vários tipos de dispositivos, como por exemplo: web, mobile, TV Digital, dentre outros.

Java também possui uma comunidade muito poderosa. Com isso, conseguimos obter muitas dicas e informações que pode nos ajudar nos estudos e a resolver problemas do dia a dia.
