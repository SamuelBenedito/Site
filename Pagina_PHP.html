O que é PHP?

PHP é uma linguagem muito popular que está entre as 5 mais utilizadas do mundo, o mesmo roda do lado do servidor e permite criar sites WEB dinâmicos, possibilitando uma interação com o usuário através de formulários, parâmetros da URL e links. Com o PHP é possível interagir com bancos de dados e aplicações existentes no servidor com a vantagem de não expor o código fonte para o cliente.




Curiosidades


Uma linguagem para desenvolvimento Web: Desde o princípio, o PHP foi desenhado como uma linguagem de script para ser usada em servidores, ou seja, foi feita com o desenvolvimento web em mente. O código PHP pode ser incluído diretamente junto como código HTML ou combinado vários frameworks existentes.



Utilizado por várias empresas grandes: Você vai encontrar esta tecnologia em vários sites, dentre eles: Facebook (o site de rede social por excelência), Wikipedia (maior enciclopédia online do mundo), Tumblr (uma popular plataforma de blog), Slack (uma ferramenta de comunicação em equipe, usado por nós aqui na Platzi), MailChimp (líder em marketing por e-mail), dentre várias outras. E você, vai começar a usar quando?



Uma linguagem livre: O PHP é uma linguagem livre e de código aberto, ou seja, você pode usá-la gratuitamente e ainda consultar e até contribuir para o código se assim desejar. Há uma grande comunidade ao redor do PHP que contribui não só com código, mas integrando em frameworks, escrevendo bibliotecas, entre outras coisas. O PHP consta de módulos para acessar servidores FTP e vários bancos de dados como PostgreSQL, MySQL, Microsoft SQL Server e SQLite.




O Que é possível criar com o PHP?

Literamente qualquer coisa. O PHP é focado principalmente nos scripts do lado do servidor, portanto, você pode fazer qualquer coisa que outro programa CGI pode fazer: coletar dados de formulários, gerar páginas com conteúdo dinâmico ou enviar e receber cookies.
